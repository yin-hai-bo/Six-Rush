# 图片资源目录

此目录用于存放游戏使用的图片资源。

## 需要的资源

### 1. 棋盘背景 (`board_bg.png`)
- **用途**：棋盘背景图
- **建议规格**：
  - 尺寸：512x512 或 1024x1024
  - 格式：PNG
  - 内容：木纹质感，3x3格子（4x4交叉点）的棋盘
  - 风格：传统围棋棋盘样式

### 2. 黑子 (`black_stone.png`)
- **用途**：黑色棋子
- **建议规格**：
  - 尺寸：128x128 或 256x256
  - 格式：PNG（带透明背景）
  - 内容：圆形围棋黑子，有立体感

### 3. 白子 (`white_stone.png`)
- **用途**：白色棋子
- **建议规格**：
  - 尺寸：128x128 或 256x256
  - 格式：PNG（带透明背景）
  - 内容：圆形围棋白子，有立体感

## 推荐资源来源

1. **OpenGameArt** (opengameart.org) - 免费游戏素材
2. **Kenney.nl** - 免费游戏资源
3. **Pixabay** - 免费图片
4. **自己绘制**：使用 Photoshop、GIMP 等工具

## 使用说明

1. 将图片文件放入此目录
2. 修改 `src/ui/board_view.rs` 使用图片资源替代代码绘制
3. 重新编译程序
