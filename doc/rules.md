# “六子冲” 游戏规则

## 1. 游戏概述

本游戏为双人对弈棋类游戏，由黑方与白方轮流行棋。双方各执6枚无等级区分的同色棋子，在4×4交叉点构成的棋盘上进行对抗。

## 2. 棋盘定义

- 棋盘由4条等距横线与4条等距纵线正交构成，形成3×3方格区域。
- 棋子仅可置于16个线段交叉点上，形成4行×4列的落子点阵。
- 采用笛卡尔坐标系定义位置：
  - 原点(0,0)位于棋盘左下角交叉点
  - x轴正方向向右延伸，取值范围{0,1,2,3}
  - y轴正方向向上延伸，取值范围{0,1,2,3}
  - 任意落子点坐标表示为(x,y)，其中x,y∈{0,1,2,3}

## 3. 棋子定义

- 每方拥有6枚完全相同的棋子，仅以颜色区分归属：
  - 黑方：6枚黑色棋子
  - 白方：6枚白色棋子
- 所有棋子在功能、移动能力及价值上完全等同，无等级或类型差异。

## 4. 初始布局

游戏开始时，棋子按以下坐标位置摆放：

|      | 棋子位置坐标                             |
| ---- | ---------------------------------------- |
| 黑方 | (0,0), (1,0), (2,0), (3,0), (0,1), (3,1) |
| 白方 | (0,3), (1,3), (2,3), (3,3), (0,2), (3,2) |

> 注：黑方位于棋盘y=0与y=1行（底部区域），白方位于y=2与y=3行（顶部区域），双方布局关于水平中线y=1.5呈轴对称。

## 5. 行棋规则

### 5.1 行棋顺序

- **黑方先行**，白方后行。
- 双方交替轮流行棋，每次只能移动一枚己方棋子。
- **不允许弃权**：轮到某方行棋时，该方必须进行合法移动，不得主动弃权或跳过回合。

### 5.2 移动规则

#### 5.2.1 移动方向与距离
每次行棋移动一枚己方棋子，移动方向仅限于**水平方向（X轴）**或**垂直方向（Y轴）**，移动距离为**一格**。

设棋子移动前的位置为 `(x, y)`，则移动后的位置 `(x', y')` 满足以下条件之一：
- **水平移动**：`(x', y') = (x ± 1, y)`
- **垂直移动**：`(x', y') = (x, y ± 1)`

即仅允许向**上、下、左、右**四个正交方向移动一格。

#### 5.2.2 移动限制
- **目标位置必须为空**：棋子只能移动到没有棋子的交叉点上。
- **不能移出棋盘**：移动后的坐标必须在棋盘范围内，即 `x' ∈ {0,1,2,3}` 且 `y' ∈ {0,1,2,3}`。
- **不能斜向移动**：不允许沿对角线方向移动。
- **不能跳跃**：不允许跳过其他棋子或一次移动多格。

### 5.3 行棋示例

假设黑方某棋子位于 `(1, 1)`，则该棋子可移动至以下位置（需满足目标位置为空且在棋盘内）：
- `(2, 1)` — 向右移动
- `(0, 1)` — 向左移动
- `(1, 2)` — 向上移动
- `(1, 0)` — 向下移动

## 6. 吃棋规则

### 6.1 吃棋时机

每次移动棋子后，立即进行吃棋判定。**本方**：指刚完成棋子移动的这一方。

### 6.2 吃棋条件

若同时满足以下条件，则吃掉对方棋子：

1. **存在共线关系**：在某一横线或直线上，存在连续排列的棋子；
2. **两枚本方棋子**：该直线上有**连续两枚本方棋子**（相邻无间隔）；
3. **刚移动的棋子参与**：这两枚本方棋子中，**有一枚是刚完成移动的那枚**；
4. **紧邻对方棋子**：这两枚连续的本方棋子**紧邻着一枚（且仅一枚）**对方棋子；
5. **形成三连**：上述三枚棋子（两枚本方 + 一枚对方）在直线上连续排列。

满足以上条件时，**该枚对方棋子被吃掉**（移出棋盘）。

### 6.3 吃棋模式图示

以下排列均在一条直线上（横线或纵线），`●` 表示本方棋子（含刚移动的），`○` 表示对方棋子：

```
● ● ○    或    ○ ● ●
↑(刚移动)           ↑(刚移动)
```

即形成「本方-本方-对方」或「对方-本方-本方」的连续三格排列时，对方棋子被吃。

### 6.4 多向吃棋

若一次移动后在多个方向（横/纵）同时满足吃棋条件，**所有满足条件的对方棋子均被吃掉**。

### 6.5 单子特殊规则

当某一方**仅剩余一枚棋子**（称为**单子方**）时，吃棋规则发生变化：

#### 6.5.1 原规则失效
单子状态下，上述第 6.2 节的「二比一」吃棋规则对双方均**不再适用**。

#### 6.5.2 优势方获胜方式
**优势方**（棋子数量多于单子方的一方）**无法通过吃子消灭单子**，只能通过**困毙**（使单子方无合法移动）来获胜。

#### 6.5.3 单子方的「担」吃规则
单子方完成行棋后，若其唯一棋子**在某一横线或纵线方向上，两侧各紧邻一枚（且仅一枚）对方棋子**，形成「对方-单子-对方」的连续三格排列，则**这两枚对方棋子被「担」掉**（同时移出棋盘）。

**图示**（`●` 为单子方棋子，`○` 为对方棋子）：

```
○ ● ○
  ↑(单子刚移动至此)
```

- 单子位于中间，左右（或上下）各紧邻一枚对方棋子
- 满足条件时，两枚 `○` 同时被吃掉

## 7. 胜负判定

### 7.1 困毙判负

若轮到某方行棋时，该方**所有己方棋子均无合法移动位置**（即全部被围困），则该方判负，棋局立即结束，对方获胜。

> 注：困毙与棋子数量无关，即使己方仍有较多棋子，只要所有棋子都无法移动，即告负。

### 7.2 无子判负

若某方棋子被全部吃掉（剩余0枚），则该方判负。

### 7.3 平局判定

当双方剩余棋子数量均**不超过2枚**时，棋局自动判定为**平局**。

> **推理依据**：
> - 1对1：单子规则下双方均无法吃子，也无法困毙对方，必为平局。
> - 2对1：单子方无法被担吃（优势方两子相连），优势方也无法困毙单子（单子灵活可逃），必为平局。
> - 2对2：任何一方吃子后变为2对1，由上述推理亦为平局。
>
> 因此，当双方棋子数均 ≤2 时，任何局面均无法决出胜负。
